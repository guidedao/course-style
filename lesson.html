Метод document.querySelectorAll() возвращает все элементы на странице.

Метод document.querySelectorAll() возвращает все отвечающие запросу элементы на странице. В качестве аргумента можно использовать любой CSS-селектор, комбинацию селекторов или даже список из нескольких селекторов через запятую.

// Получить все элементы с классом .rapper
let allRappers = document.querySelectorAll('.rapper');

// Получить все элементы с атрибутом [data-gang]
let allGangs = document.querySelectorAll('[data-gang]');


Метод document.querySelector() возвращает первый подходящий элемент на странице.

// Первый div
let firstDiv = document.querySelector('div');

// Первый div с классом .rapper
var firstRapper = document.querySelector('.rapper');

// Первый div с дата-атрибутом gang равным bloods
var firstGang = document.querySelector('[data-gang="bloods"]');

// Элемент, которого не существует
var searchWarrant = document.querySelector('.drugs');


Если элемента не существует, querySelector() вернет null. При попытке сделать что-то с несуществующим элементом, консоль выдаст ошибку. Перед использованием элемента не лишним будет убедиться в том, что он найден.

// Убедиться в наличии элемента перед обращением к нему
if (searchWarrant) {
   // Предпринять меры...
}


У каждого элемента есть чрезвычайно полезный в ряде случаев метод matches(), которым можно проверить, будет ли элемент найден с помощью селектора или комбинации селекторов. Он возвращает  true, если элемент мэтчится и  false — если нет.  

let bobbyShmurda = document.querySelector('#hotnigga');
if (bobbyShmurda.matches('.snitch')) {
   console.log('Зашквар!');
} else {
   console.log('Флекс!');
}


Существует еще целый ряд методов, позволяющих таргетить элементы конкретного типа. Но большинство из них бесполезны, поскольку всегда удобнее воспользоваться более универсальными querySelector() и querySelectorAll().

К этим методам относятся:
getElementById() — возвращает уникальный элемент по указанному айди.
getElementsByName() — возвращает нодлист элементов по параметру name.
getElementsByTagName() — возвращает элементы, найденные по HTML-тегу.
getElementsByClassName()  — возвращает элементы по названию CSS-класса. 


ПОВТОРИМ

  • querySelectorAll() используется, когда нужно найти все отвечающие запросу элементы на странице.
  • querySelector() используется, когда нужен только первый элемент.
  • Самая большая трудность — определиться с тем, нужны ли все элементы или только первый.
  • Остальные методы использовать не рекомендуется, но знать о них не помешает.
  • Все методы принадлежат объекту document и вызываться сами по себе не могут — только через точку.
